# ControlInputBox API

## 아이콘이 있는 ControlInputBox를 사용하는 경우

```tsx
    <QueueControlInputBoxComponent.IconInput valueChangeEvent={inputValueChanges} unit='degree' />
```


## 아이콘이 없는 ControlInputBox를 사용하는 경우

```tsx
    <QueueControlInputBoxComponent.Input valueChangeEvent={inputValueChanges} unit='degree' />
```

흠 하나로 다 처리되도록 변경하는게 좋을듯
