# ControlInputBox API

## 아이콘이 없는 ControlInputBox를 사용하는 경우

```tsx
<QueueControlInputBoxComponent.Wrapper color='grey' variant='filled'>
    <QueueControlInputBoxComponent.InputBox unit='percent' valueChangeEvent={inputValueChanges}></QueueControlInputBoxComponent.InputBox>
</QueueControlInputBoxComponent.Wrapper>
```


## 아이콘이 있는 ControlInputBox를 사용하는 경우

```tsx
<QueueControlInputBoxComponent.Wrapper color='grey' variant='filled'>
    <QueueControlInputBoxComponent.PrefixIcon prefixType='img' prefixValue='type'></QueueControlInputBoxComponent.PrefixIcon>
    <QueueControlInputBoxComponent.InputBox unit='percent' valueChangeEvent={inputValueChanges}></QueueControlInputBoxComponent.InputBox>
</QueueControlInputBoxComponent.Wrapper>
```
